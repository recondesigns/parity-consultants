<!-- Getting started -->
## Getting Started

### Installation

#### Clone the repo
   ```sh
   gh repo clone recondesigns/creekbridge-capital-llc
   ```

#### Install packages
   ```sh
   npm install
   ```

#### Run development server
  ```
    npm run dev
  ```

#### Storybook
  ```
    npm run storybook
  ```
